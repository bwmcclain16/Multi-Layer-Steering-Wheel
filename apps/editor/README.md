# Editor App

Runnable Python visual editor application.

Current capabilities:
- edit project metadata visually
- edit signals visually
- edit widgets visually with a page-canvas preview
- edit pages visually
- edit alerts visually
- edit profiles visually
- inspect/export live JSON
- validate and save project files


Launch:
- `./bin/mlsw-editor`
- `bin\mlsw-editor.bat`
